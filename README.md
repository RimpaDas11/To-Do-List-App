# To-Do List App ✅

A simple, responsive **To-Do List Application** built with **HTML, CSS, and JavaScript**.  
This project allows users to add, edit, delete, and mark tasks as completed. It also supports **priority levels**, **due dates**, and **persistent storage** using LocalStorage.

---

## 🚀 Features
- ➕ Add new tasks  
- ✔️ Mark tasks as complete/incomplete using a checkbox  
- ✏️ Edit existing tasks  
- ❌ Delete tasks  
- ⚠️ Validation (prevents adding empty tasks, shows error message)  
- 💾 LocalStorage support (tasks are saved even after refresh)  
- 📱 Responsive design (works on mobile & desktop)  
- ⭐ Optional features:  
  - Task priority (Low / Medium / High)  
  - Task due date  

---

## 🛠️ Technologies Used
- **HTML5** → Structure of the app  
- **CSS3** → Styling and responsive design  
- **JavaScript (Vanilla)** → Functionality and LocalStorage
